# LectureGL

**Образовательная платформа для просмотра лекций с рендерингом через OpenGL**

## Описание

Java пет-проект, представляющий собой простое приложение для просмотра образовательных материалов (лекций). Особенность проекта — вся отрисовка интерфейса выполняется через OpenGL с использованием библиотеки LibGDX и LWJGL.

Приложение позволяет отображать структурированные учебные материалы по математике и физике с поддержкой форматированного текста, изображений и интерактивной навигации.

## Технологии

- **Java** — основной язык программирования
- **LibGDX 1.9.13** — игровой фреймворк для работы с OpenGL
- **LWJGL** (Lightweight Java Game Library) — низкоуровневая библиотека для OpenGL
- **OpenGL 3.0+** — графический API для рендеринга
- **Gradle** — система сборки проекта
- **FreeType** — рендеринг шрифтов

## Формат контента

Лекции хранятся в файлах с расширением `.theory` и поддерживают следующие элементы:

### Теги форматирования
- `{{HEADER}}текст{{HEADER}}` — заголовки
- `{{MAIN}}текст{{MAIN}}` — основной текст
- `{{DEFINITION}}текст{{DEFINITION}}` — определения и важные понятия

### Изображения
```
{{PIC}}
width=200
height=200
location=./resources/Maths/Chapter1/image.png
{{PIC}}
```

### Структура курсов
- `SubSections.theory` — оглавление разделов
- `paragraphs.theory` — список параграфов в разделе
- `pN.theory` — содержимое параграфа N

## Запуск проекта

### Требования
- Java 8 или выше
- Gradle (или использовать встроенный Gradle Wrapper)
- Поддержка OpenGL 3.0+

### Сборка и запуск

**Windows:**
```bash
gradlew desktop:run
```

**Linux/macOS:**
```bash
./gradlew desktop:run
```

### Сборка JAR-файла
```bash
gradlew desktop:dist
```

Готовый JAR-файл будет находиться в `desktop/build/libs/`

## Особенности рендеринга

### Шейдеры
Проект использует custom GLSL шейдеры для рендеринга:
- **FigureShader** — отрисовка геометрических фигур (прямоугольники, линии)
- **TextureShader** — отрисовка текстур и изображений

### Интерфейс
- Полностью кастомная отрисовка UI через OpenGL
- Поддержка TrueType шрифтов через FreeType
- Интерактивные элементы (кнопки, ссылки)
- Адаптивная компоновка контента

## Функциональность

- Просмотр структурированных лекций
- Навигация по разделам и параграфам
- Отображение текста с форматированием
- Встроенные изображения
- Интерактивные ссылки между разделами
- Кнопка "Назад" для навигации
- Цветовые схемы интерфейса (Sky Blue и др.)

## Зависимости

Основные библиотеки (из [`build.gradle`](build.gradle)):
- `com.badlogicgames.gdx:gdx:1.9.13` — LibGDX core
- `com.badlogicgames.gdx:gdx-backend-lwjgl` — LWJGL backend
- `com.badlogicgames.gdx:gdx-freetype` — FreeType для шрифтов

## Конфигурация

Основные параметры в [`DesktopLauncher.java`](desktop/src/com/project/desktop/DesktopLauncher.java):
- **Заголовок окна**: "Educational Platform"
- **OpenGL**: версия 3.0 (GLES 3.0)
- **Аудио**: отключено
- **FPS в фоне**: неограничен при паузе
